v0.4.3-states Jun 20, 2026 Latest

Page-based topbar breadcrumbs & panel polish

Iteration on v0.4.2 app shell header logic: hierarchical breadcrumbs without duplicate page titles, plus a fix for double borders on nested empty states inside panels.

Topbar Breadcrumbs Empty states Panel polish
Update log
  • Topbar: page-based breadcrumb hierarchy — path in .bc, current page in .tb-title only (no duplicate titles)
  • Dashboard: workspace name in breadcrumb; main title = Dashboard
  • Tool pages: Tools / breadcrumb + tool name as main title (Social Media, Photo Tool)
  • Nested pages: full path in breadcrumb (e.g. Tools / Photo Tool /) + leaf title below (Job history)
  • Fixed overlapping borders on dashboard Connected accounts empty state — strip frame from .panel > .empty-state
