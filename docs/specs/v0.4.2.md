## Foundations

Tokens live in `design-tokens.json` (brand colors still TODO). The v0.4.2 prototype inlines a zinc neutral scale + calm blue accent (`#2563eb` / `--a-600`).

| Token group | Used for |
|---|---|
| `color.neutral` | Page/surface whites, `--n-100` fills, `--n-200`/`--border` hairlines, text `--n-500`–`900` |
| Accent (`#2563eb` placeholder) | Primary buttons, active nav, links, focus rings, active filter chips |
| `color.semantic` | Status pills, validation, toasts, trend badges |
| Spacing 4–64px | Gaps, padding, gutters |
| `borderRadius` sm–xl | Cards, inputs, buttons (6–16px in prototype) |
| `shadow` xs–xl | Cards, modals, Proto FAB |

Typography: **Inter** system stack for UI; headings 600–700, tight letter-spacing. See `DESIGN.md` for OKLCH target tokens (Plus Jakarta Sans for marketing surfaces).

## Account & onboarding state machine

Prototype wires these transitions (production should mirror):

| Trigger | Effect |
|---|---|
| Signup → Dashboard | Starts in **New** account state |
| Proto FAB · Active | Dashboard populated · Social has posts · Photo full pipeline |
| Connect channel | Social → **no posts** · show tabs/chrome |
| Schedule post (composer) | Account → **Active** · Social → **active queue** · toast |
| Save draft | Toast only |
| Photo dropzone / browse | Photo → full New job flow · toast |
| Proto FAB · New | Resets dashboard empty · social no channels · photo empty |

### Social states summary

1. **No channels** — full empty state; no tabs.
2. **No posts** — tabs visible; Publishing to chips; queue empty state.
3. **Active** — populated queue grouped by day.

## Cross-cutting states & patterns

- **Loading:** skeleton rows/cards for lists; spinners on buttons only.
- **Toasts:** centered bottom, ~2.6s, for composer save/schedule, photo upload, etc.
- **Errors:** inline panel alert + toast for transient failures.
- **Empty states:** icon + headline + one line + single primary action (+ optional secondary link).
- **Destructive actions:** confirm dialog (delete post, watermark, disconnect).
- **Filter chips:** toggle; "All" clears others; active = `--a-50` bg, `--a-700` text, `--a-500` border.
- **Modals:** backdrop click closes; Esc closes modals, search, caption drawer.
- **Responsive:** sidebar → icons <1024px; hamburger <768px; composer panes stack with preview tab on mobile.
