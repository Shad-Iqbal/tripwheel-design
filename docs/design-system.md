# Design System — Tripwheel

## Brand

- **Primary**: `oklch(52% 0.14 250)` — deep confident blue
- **Primary foreground**: `oklch(98% 0.005 250)` — near-white on primary
- **Accent**: `oklch(62% 0.18 250)` — lighter active blue
- **Background**: `oklch(98.5% 0.004 250)` — cool off-white
- **Surface**: `oklch(100% 0 0)` — pure white cards
- **Surface raised**: `oklch(99% 0.003 250)` — subtle lift
- **Border**: `oklch(91% 0.008 250)` — cool gray border
- **Border subtle**: `oklch(95% 0.005 250)` — barely-there divider
- **Muted**: `oklch(95% 0.006 250)` — backgrounds for chips/badges
- **Text primary**: `oklch(16% 0.02 250)` — near-black, tinted cool
- **Text secondary**: `oklch(45% 0.015 250)` — medium gray
- **Text tertiary**: `oklch(62% 0.01 250)` — light labels
- **Success**: `oklch(58% 0.16 145)`
- **Warning**: `oklch(72% 0.18 75)`
- **Danger**: `oklch(55% 0.2 25)`
- **Info**: `oklch(58% 0.14 250)` — matches primary family

## Fonts

- **Display / Headings**: `'Plus Jakarta Sans', sans-serif` (600–700)
- **Body**: `'Inter', sans-serif` (400–500)

- **Google Fonts**: `https://fonts.googleapis.com/css2?family=Plus+Jakarta+Sans:wght@500;600;700;800&family=Inter:wght@400;500;600&display=swap`

- sm: `0 1px 2px oklch(0% 0 0 / 0.05)`
- md: `0 4px 12px oklch(0% 0 0 / 0.06)`
- lg: `0 12px 32px oklch(0% 0 0 / 0.08)`

## Tone

Clean, focused, like Linear. Information-forward but calm. High contrast where it matters (CTAs, status), muted everywhere else. Generous whitespace between sections, tight inside components. No decorative elements — every pixel earns its place.
