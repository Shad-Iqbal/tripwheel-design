Two directions: Minimal vs Structured
Initial exploration of the Social Media queue. Minimal takes a Linear-like approach — borderless post rows, generous whitespace, hover-revealed actions, understated filters. Structured goes Stripe-like — posts in a paneled container, always-visible actions, stats bar, sidebar widget showing next post countdown. Both use the same OKLCH token set with Plus Jakarta Sans + Inter.
